Project information

  • 주제: Korea Dessert 판매를 위한
    Singapore region On-premise 환경 구축
  • 프로젝트 기간: 2024년 2월 5일 ~ 2월 29일
  • 핵심목표: VMware ESXi를 활용한 HA 서비스 구축
  • 담당 역할: Project Leader
    VMware를 활용한 가상화 스토리지 서버 구축, DNS(이중화), NFS서버 구축, 3-Tier 구축 Assistance

프로젝트 목표

  • 인프라 및 서비스의 이중화를 통해 HA 확보
  • VMWARE ESXi를 사용한 가상서버 운용
  • Zabbix를 통한 모니터링 환경 구축
  • NAT를 통한 내/외부망간 통신 구성
  • Proxy 서버를 통한 보안성과 서버 부하 분산
  • SSL 인증을 통해 웹 보안 강화
  • NFS서버를 통한 운영 ↔ DR 간 데이터 동기화

    • 배운점

    • On-Premise 구조와 구축 방법
    • 가상화구성을 위한 Storage Bios 세팅 방법
    • VMware ESXi OS 설치 방법
    • VMware vCenter 구축
    • Windows Server DNS 구축, Linux DNS구축
    • Windows Server NFS 구축, Linux NFS구축
    • Linux Nginx, PHP, MariaDB,HAProxy 구축

    어려운점

    • vSphere unsupported CPU 문제
    • Virtual OS registration fails on server VM 문제
    • Hardware not recognized 문제

    해결방법

    • 강제로 CPU가 허용되도록 값을 허용하여 해결
    • Storage Bios로 접근하여 Virtual tchnology 활성화하여 해결
    • Bios설정에서 ROM Configuration에서 Drive 재구성하여 해결
